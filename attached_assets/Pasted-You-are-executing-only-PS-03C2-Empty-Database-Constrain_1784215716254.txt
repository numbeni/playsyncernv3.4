- obsolete normalized-email unique index

- PSN Password ciphertext + keyed lookup hash
- Email Password ciphertext + keyed lookup hash
- Family Management Email ciphertext + keyed lookup hash

These canonical fields must be:

- NOT NULL
- not blank
- separately indexed where exact lookup is needed

Finalize required Account fields:

- game_id
- account_code
- account_number_prefix
- account_number_seq
- display_number
- all approved ciphertext and lookup-hash fields
- online_id
- normalized Online ID
- birth_date

- SOLD
- INACTIVE
- NULL

Require:

- account_number_seq > 0
- required text/hash/ciphertext fields are not blank
- duplicate normalized Online IDs remain allowed

Approved formats remain:

- ACC-000001
- GOW-001

- USED
- REVOKED

Require:

- USED requires used_at
- AVAILABLE and REVOKED require used_at to be NULL

Do not implement Reveal or “at least one Backup Code” service logic.

Finalize FINISHED consistency:
